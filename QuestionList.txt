h1>Special Problems 10 ->IP Address Restoration</h1>
  <p>Given a string containing only digits, restore it by returning all possible valid IP address combinations. A valid IP address consists of four fields, each field is separated by a "." and must contain a value between 0 and 255 (inclusive).</p>
  <p>Constraints:</p>
  <ul>
    <li>The string length must be between 4 and 12.</li>
  </ul>
  <p>Example:</p>
  <p>Input: "25525511135"</p>
  <p>Output: ["255.255.11.135", "255.255.111.35"]</p>
    <p>
      You can use any programming language you like. Let me know if you have any questions about the problems or if you need additional clarification.
