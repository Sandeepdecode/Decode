<h2>Problem 3</h2>
    <p>
      Write a function <code>is_prime</code> that takes a positive integer <code>n</code> and returns <code>True</code> if <code>n</code> is a prime number, and <code>False</code> otherwise.
    </p>
    <h2>Constraints</h2>
    <ul>
      <li><code>n</code> is a positive integer with value at most 10^6.</li>
    </ul>
    <h2>Examples</h2>
    <pre>
is_prime(2) => True
is_prime(3) => True
is_prime(4) => False
is_prime(5) => True
is_prime(6) => False
    </pre>
