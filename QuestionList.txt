<p><strong>Problem 1</strong></p>
<p>Write a function <strong>largest_sum</strong> that, given a list of integers <strong>nums</strong>, returns the maximum sum of a contiguous sublist of <strong>nums</strong>.</p>
<p>For example, given the list <strong>[-2, 1, -3, 4, -1, 2, 1, -5, 4]</strong>, the function should return <strong>6</strong>, because the contiguous sublist with the largest sum is <strong>[4, -1, 2, 1]</strong>, which has a sum of <strong>6</strong>.</p>
<p>You may assume that the input list has at least one positive integer.</p>
<p><strong>Constraints</strong></p>
<ul>
  <li>The input list has at most length $10^4$.</li>
  <li>The elements of the input list are integers with absolute value at most $10^9$.</li>
</ul>
<p><strong>Notes</strong></p>
<ul>
  <li>A contiguous sublist is a sequence of elements that appears in the input list in the same order that they appear in the input list.</li>
  <li>A sublist is a subset of the input list, possibly with elements removed.</li>
</ul>
<p><strong>Example</strong></p>
<pre>
largest_sum([-2, 1, -3, 4, -1, 2, 1, -5, 4])
</pre>
<p><strong>Output</strong></p>
<pre>
6
</pre>
<p>You can use any programming language you like. Let me know if you have any questions about the problem or if you need additional clarification.</p>
