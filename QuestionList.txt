    <p>
      Write a function <code>is_even</code> that takes an integer <code>n</code> and returns <code>True</code> if <code>n</code> is even, and <code>False</code> otherwise.
    </p>
    <h2>Constraints</h2>
    <ul>

    </ul>
    <h2>Example</h2>
    <pre>
is_even(2)
    </pre>
    <h2>Output</h2>
    <pre>
True
    </pre>
